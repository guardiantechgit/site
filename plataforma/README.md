# GuardianTech

GuardianTech é uma aplicação web desenvolvida para facilitar o acesso à plataforma de rastreamento da Guardian Tech. O projeto foi pensado para oferecer uma experiência moderna, responsiva e segura, com possibilidade de instalação como aplicativo (PWA) em dispositivos móveis.

## Funcionalidades
- Tela de login customizada e responsiva
- Suporte a instalação como aplicativo (PWA) em celulares
- Interface moderna e intuitiva
- Proteção contra seleção de texto e atalhos de desenvolvedor

## Estrutura do Projeto
```
index.html           # Página principal de login
style.css            # Estilos customizados
manifest.json        # Manifesto para PWA
image/               # Imagens e ícones do projeto
```

## Como instalar no celular (PWA)
1. Acesse o site pelo navegador do seu celular.
2. Um botão "Instalar app" será exibido na parte inferior da tela.
3. Clique no botão e siga as instruções do navegador para adicionar o app à tela inicial.

## Personalização
- Para alterar o logo ou ícone do app, substitua as imagens na pasta `image/`.
- As cores e estilos podem ser ajustados no arquivo `style.css`.
